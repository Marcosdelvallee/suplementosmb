# MB SUPLEMENTOS

Frontend de e-commerce premium para suplementos deportivos, construido con React, Vite, Tailwind CSS y Framer Motion.

## Requisitos

- Node.js instalado
- npm funcionando correctamente

## Instalacion

```powershell
npm install
```

## Desarrollo

```powershell
npm run dev
```

Luego abri la URL local que muestre Vite, normalmente:

```text
http://localhost:5173
```

## Build de produccion

```powershell
npm run build
```

## Estructura

```text
mb-suplementos/
  index.html
  package.json
  postcss.config.js
  tailwind.config.js
  vite.config.js
  src/
    App.jsx
    main.jsx
    components/
    data/
    styles/
```

## Secciones incluidas

- Navbar
- Hero section
- Productos destacados
- Categorias
- Ofertas
- Beneficios
- Testimonios
- Footer

## Notas

Si `npm` falla en Windows, reinstala Node.js LTS y verifica:

```powershell
node -v
npm -v
```
